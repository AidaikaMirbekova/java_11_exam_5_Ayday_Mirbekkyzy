В данной контрольной работе была создана база данных со следующими таблицами:

1)products: таблица с  товарами
в нее входят поля:
id(PK),
product,
category_id(FK), связь с таблицей categories
brand_id(FK),связь с таблицей brands
decsription;

2)categories:таблица с  товарами
в нее входят поля:
id(PK),
category;

3)brands:таблица с брендами продуктов:
в нее входят поля:
id(PK),
brand;

4)wirehouses:(склад)таблица с информациями о складах
в нее входят поля:
id(PK),
wirehouse;

5)suppliers:таблица с информациями об контрагентах
в нее входят поля:
id(PK),
supplier;

6)coming:таблица с информацией о поступлении товаров,
в нее входят поля:
id(PK),
coming_time,время приходования товара
com_product_id(FK),номер товара который приходуют / cвязь с таблицей products
qty, количество товара
com_warehouse_id(FK),номер склада куда приходуют товар / связь с таблицей warehouses
com_supplier_id(FK); при новом поступлении товара пишется номер контрагента который поставляет новый товар / связь с таблицей suppliers


7)expense:таблица с информацией о выходе со склада или выдачи  товаров контрагентам,
в нее входят поля:
id(PK),
action_time, время выхода товара со склада
ex_product_id(FK), товар  / cвязь с таблицей products
qty,количество товаров которые забрали
ex_warehouse_id(FK), номер склада откуда товар забрали  / связь с таблицей warehouses
ex_supplier_id(FK);либо если забрал контрагент,номер конртрагента,который забрал товар / связь с таблицей suppliers
